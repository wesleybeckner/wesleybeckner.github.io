We will consider the following while analyzing our data¶

  • Use split screen

    • load class_data.csv from the catalyst website into excel
    • go to view, split
    • scroll bottom pane to bottom of sheet
    • alternatively, it is much easier to use ctrl+shift+arrows and tab to navigate the excel workbook once you get the knack for it

  • Add a time series column

    • create a new column
    • select column cells down to last OD value and type alt+H+F+I+S to bring up the series menu
    • increment by 0.01 to a final value of 30
    • by the way, lets say we got our x-values wrong to properly fit our model. we can easily go back to our original filter and change those values

  • Use dual windows

    • go to insert, scatter plot. select the data then go to view, new window
    • create a new sheet and paste the graph in this window
    • you should now have two excel books that are linked, the left one containing a view of the header and footer of your data and the right one containing the graph we are working on

  • Use filters
    • select all three columns and got to data, filter
    • we can now select our researchers data one-by-one and paste them into a paste sheet
    • note: if we paste into the current sheet the values will update depending on our filter
    • lets go ahead and get four separate data columns for our researchers

  • Use solver to create a growth model
    • lets say we trust peters measurements. what sorts of analysis can we do in excel to determine whether the other researchers are taking accurate or imprecise measurements?
    • In your homework, you will have a multidimensional data set you need to explore using filters to make a rational decision on how to choose the parameters for your model
    • Use the following equation to create a model from Peter's time series:
    • y(t) = a * exp(b(exp(-ct)))
    • a = 1
    • b = - 8
    • c = 0.4
    • rss error = sqrt(sum(data-predicted)^2)
    • make sure solver is turned on and solve for a, b, c
